# GBT-Reliability-Analysis
To evaluate the reliability and statistical stability of Greater Bridgeport Transit (GBT) service at a high-traffic corridor while using Statistical Quality Control (SQC) tools such as lean six sigma and DMAIC to identify root causes and suggest data-driven improvements.
